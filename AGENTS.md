# AI Agents & Development Tools

This document describes the AI agents, tools, and development workflows used in the portfolio project.

## Development Assistants

### Cascade AI Assistant
- **Role**: Primary AI coding assistant for development
- **Capabilities**: 
  - Code generation and modification
  - Debugging and troubleshooting
  - Project architecture guidance
  - Documentation generation
- **Integration**: IDE-based assistant for pair programming

## Development Workflow

### AI-Assisted Development
- The project utilizes AI-assisted development for:
  - Rapid prototyping and feature implementation
  - Code refactoring and optimization
  - Documentation maintenance
  - Bug identification and resolution

### Best Practices
- All AI-generated code is reviewed and tested before deployment
- AI assistants are used as pair programming tools, not replacements for human oversight
- Code follows established patterns and project conventions
- Security and performance are prioritized in all AI-assisted changes

## Tooling

### Build Tools
- **Turborepo**: Monorepo build system for efficient task execution
- **Bun**: Fast package manager and runtime
- **Astro**: Framework for building static sites

### Deployment
- **Cloudflare Pages**: Static site hosting with global CDN
- **Wrangler**: Cloudflare CLI for deployment configuration

## Version Control
- **Git**: Distributed version control
- **GitHub**: Code hosting and collaboration platform
